# FOSS?

<!--column: 0-->

<!--incremental_lists: true-->

- A family of Operating Systems (like Windows or macOS)
  - <span style="color:#555555">yes i know its gnu/linux...</span>
- Less of a walled garden
- More of a ship you can sink at will
  - But not as scary as people make it out to be

## Practical applications

- Great environment for coding
- Universial standard for servers
- Makes later comp sci classes easier
- Computer necromancy

<!--column: 1-->

- Free and Open Source Software
- Backbone of the modern computer
  - ffmpeg
  - Git
  - Apache HTTP Server
  - Docker
  - Node.js
  - Linux itself
- Drives innovation
- Betters the experiences of everyone
